Codex System Rules — Agro Veterinaria Varoli
Purpose
This project uses Codex (ChatGPT) to design and build digital solutions for Agro Veterinaria Varoli and related local businesses in the Punilla Valley, such as:
Agro Veterinarias (Focus: Fernando Varoli)
Ferreterías Rurales
Corralones de Campo
Tiendas de Forrajes y Semillas
Pet Shops locales
The objective is to create simple, intuitive, modern websites in Spanish (Argentina) that are easily accessible for users in Córdoba who may not be tech-savvy.
These websites must support AI-driven inventory updates (PDF Importer) and integration with Supabase, allowing automatic stock management from the business's internal lists.
Language Rules
All interfaces must be built in Spanish (Español Argentino).
Code comments and variable names should be in English (standard engineering practice), but:
UI Text / Labels
Buttons / CTAs
Admin Panel / Dashboards
Must be Spanish-first and use local terms (e.g., "Turnos" instead of "Citas").
Execution Rules (Codex Protocol)
Codex must never generate or execute code automatically without prior confirmation.
Before programming, Codex must:
Analyze: Identify if it's for the main Vet or a demo.
Engineering Filter: Ask about Supabase tables, Edge Functions, or MP integration.
Propose: Show the technical path (Next.js App Router).
Wait: Confirm with the developer.
Code: Execute the implementation following the brand's Red/Black palette.
Multi-Agent Restriction
This system does not allow multi-agent workflows.
Only one AI agent (Codex/ChatGPT) should operate at a time to maintain context consistency and avoid conflicting architectural decisions.
Business Objective
The system must be capable of generating:
Real Production Websites: Fully functional (e.g., the current site for Fernando).
Demo Websites for Sales: Visual examples to present to other local agro-businesses.
Demos allow the expansion of the "Varoli Model" to other partners or branches before a full deployment.
Demo Generation
Codex should be able to generate visually complete and realistic templates such as:
demo-agro-veterinaria
demo-ferreteria-rural
demo-corralon-punilla
demo-forrajes-balanza
Target Users (The Varoli Profile)
Primary users of these websites are:
Fernando Varoli and his local employees.
Producers and pet owners in the La Falda region.
Therefore, the admin and client interfaces must be:
Simple & Minimal: No clutter.
Intuitive: Big buttons, clear labels.
Mobile Friendly: Optimized for touch and 4G connections.
Brand Consistent: Always using Rojo Óxido #8B1D1D.
